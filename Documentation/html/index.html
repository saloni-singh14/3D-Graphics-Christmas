3D Graphics

In this assignment we learnt about the 3D viewing pipeline and Matrix transformations. We also learnt about Matrix stack operations. We have used primitives like Cubes, Torus, Utah's Teapot, sphere, cone and cylinder to create:

  • Snowman
  • Pine Trees
  • Tables and Chairs
  • House
  • Windows
  • Carpet
  • Gifts
